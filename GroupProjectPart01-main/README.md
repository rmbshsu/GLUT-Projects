# GroupProjectPart01
First part of Computer Graphics group project.

Controls:
1) Each vertex of a user-defined polygon is defined by clicking in the viewport. The final result is determined by clicking the first vertex a second time.
2) Click and drag the polygon to move it around the scene.
3) Click and drag the polygon while holding SHIFT to increase the scale.
4) Click the polygon while holding ALT to switch the rotation direction between Clockwise and CounterClockwise.
